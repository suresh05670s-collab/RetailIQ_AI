# RetailIQ AI

## Intelligent Retail Analytics & Business Intelligence Platform

RetailIQ AI is a web-based Retail Sales Analytics and Business Intelligence platform developed using Python, Flask, SQLite, Pandas, Plotly, and Machine Learning. The system helps retail businesses manage products, customers, sales, and generate business insights through interactive dashboards and AI-powered forecasting.

---

# Project Overview

RetailIQ AI enables businesses to:

- Manage products
- Manage customers
- Record sales
- Analyze business performance
- Visualize sales data
- Generate AI insights
- Forecast future sales using Machine Learning

# Installation

Clone the repository

```bash
git clone https://github.com/suresh05670s-collab/RetailIQ_AI.git
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

---

# How to Run

1. Clone the repository
2. Install Python packages
3. Create the SQLite database
4. Train the Machine Learning model
5. Run the Flask application
6. Open:

```
http://127.0.0.1:5000
```
